1. Removing RSA key from a know IP

  • This is useful when you redid a computer and set it to the same static IP address. So if you're working from a Mac or other machine, you need to remove the old RSA key so it can be replaced with the new one from the new machine.

  • Run the following command to remove the conflicting key associated with the specific IP address:

    ssh-keygen -R <IP_ADDRESS>

  • For example:

    ssh-keygen -R 192.168.1.100
